Project Overview
This project focuses on analyzing hospital patient records to optimize healthcare delivery and operational efficiency. Using Power BI, I transformed raw medical data into an interactive dashboard that tracks patient demographics, admission trends, and departmental performance.

Key Insights & Analysis
Patient Demographics: Segmented data by age, gender, and location to understand the primary populations being served.
Admission Trends: Identified peak admission periods and average length of stay (LOS) to help with hospital staffing and bed management.
Departmental Performance: Evaluated the workload and efficiency of different hospital wings (e.g., ER, Cardiology, General Ward).
Health Outcomes: Monitored discharge types and patient recovery rates to visualize quality of care.

Tools & Techniques
Power BI: Data Modeling, DAX (Data Analysis Expressions) for custom metrics, and interactive visualizations.
Data Cleaning: Handled missing patient data and normalized categorical records for accurate reporting.
Dashboard Design: Created a user-friendly
interface for hospital administrators to filter data by date, department, and patient status.
